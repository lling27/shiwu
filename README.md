# 拾悟。 · 道次第修学日记

一个围绕《菩提道次第》闻思修的修学日记网页应用——把学修路上的心得,轻轻拾起来。

单个 `index.html` 文件,零依赖、零构建、零服务器。所有数据只保存在你自己浏览器的本地存储里,不会上传到任何地方。

## 功能

### 记录
- **快速记录**:一两句话也能记,支持 `Ctrl / ⌘ + Enter` 快速保存
- **闻 ‧ 思 ‧ 修**:每条笔记可标记类型——闻(听闻理解)、思(思维法义)、修(座上观修),不同颜色区分
- **挂到法类**:笔记可归到道次第科判的对应法类下(念死无常、皈依三宝、深信业果……)
- **引导问题**:写不出来时,从内置的修学反思题库随机抽一个(如"今天的烦恼,它的因是什么?对治它的法又是什么?"),点"就写这个"以引用形式插入正文
- **标签**:自由标签作为补充维度

### 两个视图
- **时间线**:按日期倒序浏览全部笔记,支持关键词搜索、闻思修筛选、标签筛选、法类筛选(点笔记上的法类名即可)
- **道次第**:按科判结构浏览——道前基础 → 下士道 → 中士道 → 上士道,每个法类显示笔记数量和最近记录日期,点开可查看该法类下的全部思考轨迹,并可"设为当前进度 📍"

### 科判可编辑
默认科判基于《菩提道次第略论》框架搭建。在「道次第」视图点"编辑科判",可以添加/重命名/删除大科与子目,以便对齐你实际学习的讲记章节(如济群法师《道次第》系列)。调整科判不影响已有笔记。

### 回顾与统计
- "那年今日"与"随机重读"卡片,让旧的思考不断被重新遇见
- 连续记录天数、法类覆盖度统计

### 数据归你
- 一键导出 Markdown(含闻思修与法类标注,适合阅读归档)
- 导出/导入 JSON 完整备份(含笔记、科判、进度),换设备迁移无损
- 深浅色主题,跟随系统亦可手动切换

## 使用方式

**部署自己的实例**:Fork 本仓库,在仓库 Settings → Pages 中把 Source 设为 GitHub Actions,推送到 `main` 即自动部署。

> 隐私说明:本应用没有服务器和数据库,所有日记内容只保存在你自己浏览器的 localStorage 中,不会上传到任何地方。换设备或清除浏览器数据前,记得用"备份 → 导出 JSON"保存一份;在新设备上"导入 JSON 备份"即可恢复。请勿将导出的备份文件提交到公开仓库。
